Formularze zbierają i wysyłają określone dane w celu złożonym przez autora. Wprowadzone dane można przesłać na adres poczty mailowej lub wykorzystać w uruchomionym skrypcie zarówno po stronie klienta, jak i zdalnego serwera.

Każdy formularz mieści się między znacznkami form
    Podstawowe znaczniki:
  1. input
  2. select
  3. textarea
  4. fieldset
  5. legend
  6. label
    podstawowe atrybuty (type)
  1. text
  2. password
  3. radio
  4. checkbox
  5. submit
  6. reset
  7. file
  8. image
  9. mail
  10. url
  11. number
  12. data
  13. time
  14. week
  15. month
  16. tel
  17. search
  18. color
  19. range
    Pozostałe atrybuty
  1. name
  2. value
  3. action
  4. method="get/post"
  5. maxlength="x"
  6. enctype
  7. rows="x"
  8. cols="x"
  9. required
  10. pattern
  11. submit
  12. multiple
  13. datalist
  14. fieldset
    Inne atrybuty
  1. placeholder
  2. autocomplete
  3. autofocus
Atrybut niesprawdzający danych: novalidate (form action="index.php" novalidate)
